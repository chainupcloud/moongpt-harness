## 交易流程

```
① Connect Wallet
   → Web3Modal 弹窗（MetaMask / WalletConnect / Binance / 540+）
   → 连接到 Sepolia 测试网（chainId: 11155111）

② Deposit（充值）
   → Bridge USDC 到 Hyperliquid L1（最低 $10）
   → 资金从 EVM 链转至 Hyperliquid L1

③ Trade（下单）
   → 选择市场（测试网当前仅 BTC-USDC）
   → 设置 Cross/Isolated Margin，最高 50x（BTC 最高 20x）
   → 订单类型：Market / Limit / Stop / TWAP / Scale / Scalp / Trailing Stop
   → 钱包签名 → Hyperliquid L1 执行（<200ms）

④ Withdraw（提现）
   → 从 Hyperliquid L1 提回 EVM 链
```

## 发现的问题

### P1 — /app 路径 404
- **现象**：访问 `https://moongpt.ai/app` 返回 404
- **影响**：若有外链引导到 `/app`，用户无法访问
- **建议**：添加 redirect 到 `/trade`

### P2 — 多页面共用同一 `<title>`
- **现象**：`/markets`、`/swap`、`/predict`、`/positions` 标题均为 "Hermes | Trade everything, spend everywhere"
- **影响**：SEO 差，浏览器 tab 无法区分页面
- **建议**：各页面设置独立 `<title>`

### P3 — /dashboard 内容为空
- **现象**：`/dashboard` 页面加载但无内容渲染
- **需验证**：连接钱包后是否有内容显示

### P4 — 测试网 BTC-USDC 价格异常偏低 / 流动性
- **现象**：测试网仅有 BTC-USDC 一个市场，价格约 $70,071
- **影响**：测试覆盖面有限，无法测试多市场场景
